new * Solves equations of the form a * x = b(options)

This is the main Class used to combine dom elments and to listen for stream events.

The ViewStream class attaches one dom element to one another by passing a reference to one anothers rxjs obxerver.

The ViewStream object can create a dom elment that is as simple as a <p> tag with text or as complex view based on a Mustache template and a json object
