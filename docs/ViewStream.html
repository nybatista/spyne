ViewStream(props)

new ViewStream(props)

The ViewStream object creates or references a dom elment; the dom element can be as simple as a <p> tag, or the dom element can be a large dom chunk that is generated from data and an html template.

Generally, the idea is to extend the ViewStream class and add the custom values within the constructor function, but all of the values can be passed as parameters as well.

addChannel(str, bool, bool)

Preferred method to connect to instances of registered channels, such as 'DOM', 'UI', and 'ROUTE' channels.

Add Channel will automatically unsubscribe to the channel, whereas the getChannel method requires the developer to manually unsubscribe.

Parameters:
Name Type Default Description
str string

The name of the registered Channel that was added to the Channels Controller.
bool boolean false

false, add true if the View should wait for this channel to unsubscribe before removing itself.
