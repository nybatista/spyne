Class: DomItem

DomItem(tagName, attributes, content, template)

new DomItem(tagName, attributes, content, template)

This class is mostly used internally by the ViewStream object, but it can be also used to generate a lightweight dom element that contains no props or parameters.

Parameters:
Name Type Default Description
tagName string div

the tagname for this dom element.
attributes object

any domElement attribute (except for class )
content string | object

string for text tags and json for templates
template tmpl
